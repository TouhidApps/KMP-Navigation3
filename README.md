# KMP Navigation3 + JetpackCompose Navigation3
KMP-Navigation3 with animation and organized implementation. Also included jetpack compose navigation 3 with xml based old project implementation - all in one navigation 3

## Project 1 - JetpackCompose-Navigation3

### Features-

- Old xml based project support
- New full fresh compose project support
- Well arranged navigation route
- Data/Object transfer to route
- BackHandler

## Project 2 - KMP-Navigation3

### Features-

- Compose Multiplatform with Android/iOS/Desktop support
- Well arranged navigation route
- Data/Object transfer to route
- Landscape orientation support (Split screen)
- List and detail view support
- Alert dialog support
